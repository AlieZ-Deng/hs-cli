# hs-cli

# 这是一个测试 cli 工具

用以快速生成 react + express + ssr 项目

------



# 安装

```
# npm
$ npm install -g zhurong-cli

# or yarn
$ yarn global add zhurong-cli
```

# 使用

```
$ hs create <dirname>

# dev 
$ npm run dev

# prod
$ npm run build
$ npm run server
```
